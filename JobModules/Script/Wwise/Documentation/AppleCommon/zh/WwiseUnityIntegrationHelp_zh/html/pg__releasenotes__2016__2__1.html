版本说明 - Wwise Unity Integration 2016.2.1

此次发布为 2016.2.1 提供了 Wwise SDK 更新。

兼容性：

  • Wwise SDK: 2016.2.1
  • Unity: Unity 5.5（Personal 或 Pro）。
    		备注： 重要提示：该集成还没有在 Unity 4 上测试过。
    平台： Wwise Unity Integration 在以下平台测试过：
    Android
    iOS
    Linux
    Mac
    PS4
    PS Vita
    tvOS
    Windows
    Windows Store (Universal Windows Platform)
    Xbox One     		Unity 5.5.0f3
    Wii U Unity 5.4.2f2

已解决的问题：

  • WG-27085：减少了 AuxSends 造成的分配
  • WG-31127：为 AK_MusicSyncUserCue 修复了返回的用户提示名。
  • WG-31650：修复了当声音引擎未被初始化时 AkGameObj 中的 NullReferenceException。
  • WG-31651：避免在调用 SetObjectPosition 时自动创建 AkGameObj。
  • WG-31862：从事件回调列表中删除与事件回调无关的枚举值。

已知存在的问题

  • Unity case #861189：如果安装了 Unity Windows Store 播放引擎，使用 Launcher 来安装就会失败。
