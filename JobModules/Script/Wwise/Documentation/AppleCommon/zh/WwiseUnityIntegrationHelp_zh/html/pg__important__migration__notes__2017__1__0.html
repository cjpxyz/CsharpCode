重要迁移说明 2017.1

针对 AkCallbackManager 有以下重大更改：

  • AkCallbackManager.AudioInterruptionCallback() 的签名更新了： 
    public delegate AKRESULT AudioInterruptionCallback(bool in_bEnterInterruption, object in_Cookie);
  • AkCallbackManager.BGMCallback() 的签名更新了: 
    public delegate AKRESULT BGMCallback(bool in_bOtherAudioPlaying, object in_Cookie);
  • AkCallbackManager.EventCallback() 的签名更新了: 
    public delegate void EventCallback(object in_cookie, AkCallbackType in_type, AkCallbackInfo in_info);
  • 回调类不再嵌套在 AkCallbackManager 中，并且有一个从 AkCallbackInfo 继承了许多内容的层级结构。只要有可能，这些类的关联和成员会模仿其 C++ 版本的语法。
  • 回调类的 gameObjID 成员现在是 ulong 类别，而不是 IntPtr。
  • AkCallbackManager.AkMidiEventCallbackInfo 这一类现在是 AkMIDIEventCallbackInfo。（请注意 "MIDI" 的大写）
  • AkCallbackManager.AkMusicSyncCallbackInfoBase 这个类被移除了。现在变为使用 AkMusicSyncCallbackInfo 这个类。
  • AkCallbackManager.AkMusicSyncCallbackInfo.segmentInfo 的所有成员已被开放成为 AkMusicSyncCallbackInfo.segmentInfo_* 的成员了。
