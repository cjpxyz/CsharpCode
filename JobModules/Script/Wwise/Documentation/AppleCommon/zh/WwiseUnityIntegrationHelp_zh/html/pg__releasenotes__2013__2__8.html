版本说明 - Wwise Unity Integration 2013.2.8

此次发布为 Wwise 2013.2.8 提供了 Wwise SDK 更新，新的 Wwise-Unity 集成工具，新增文档和错误修复。

显著功能：

  • 更加简单的新工程设置流程（参见 在 Unity 工程中安装或升级集成）
  • 使用 Wwise Picker 访问 Wwise 工程更加容易（见 Wwise Picker）
  • 新的和改进的 Unity 组件，用于封装常见的音频任务（见 使用 Wwise Unity 集成）
  • Wwise-Unity Integration 文档合并，并和 Wwise SDK文档（就是本文档哦！）交叉引用

兼容性：

  • Wwise SDK: 2013.2.8
  • Unity: 4.x

支持的平台：

  • Android
  • iOS
  • Linux*
    • 32 位
    • 64 位
  • Mac OS X（10.6 和更高版本）
  • PS3
  • PS4
  • PS Vita
  • Windows
    • 32 位
    • 64 位
  • Windows 8（Metro）
    • Desktop
    • 在 Intel 设备上的 Windows Store App，和 ARM 平板电脑
  • Windows Phone 8.0
  • Xbox360
  • Xbox One

新功能：

  • 添加了对 Xbox One 平台的支持。
  • 添加了对 PS4 平台的支持。
  • 添加了对 Linux 平台的支持。* Wwise 2013.2.8 Linux BETA 版支持 Linux 的 Unity Integration。
  • 添加了对 Windows Phone 8.0 平台的支持。

迁移：

  • 现在，Multi-architecture Windows 和 Metro 插件安装到了它们指定的文件夹中，并且不再需要使用 Unity 的 Scripting Define Symbol 功能来切换。
  • 为了简单起见，Unity Editor 菜单中重新构建 Integration 的功能被移除。
  • 将现成的 Integration 与源代码分离。
  • 将 Windows IDE 解决方案从 Visual Studio 2008 升级到 2010。
  • 更新了 Windows 和 Metro 的指定平台文件夹。
