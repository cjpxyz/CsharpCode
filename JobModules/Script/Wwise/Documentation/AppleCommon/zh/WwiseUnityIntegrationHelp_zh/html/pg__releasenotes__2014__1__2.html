版本说明 - Wwise Unity Integration 2014.1.2

此次发布为 Wwise 2014.1.2 提供了 Wwise SDK 更新，并且修复了错误。

兼容性：

  • Wwise SDK: 2014.1.2
  • 由于 XDK 在 Unity Editor（10 月 XDK）和 Wwise SDK（11 月 XDK）之间不兼容，因此 Xbox One 平台无法使用此版本。

已解决的问题：

  • WG-26305 修复了：使用 null GameObject 在 64 位构建中报告一个未注册的 GameObject。
  • WG-26337 修复了：增加了对 Mac 64 位平台的兼容性。
  • WG-26385 修复了：Wwise 在 Android 系统的背景下在 Unity 中占用 CPU。
  • WG-26395 修复了：修复了下述情形下的构建失败：从命令行使用 MSBUILD 构建 Windows Phone 8 工程时。
  • WG-26430 修复了：修复了在 iOS 上调用 SetBasePath() 时的崩溃。

已知问题：

  • WG-25669：Wwise Picker 的 Auto Populate 在 Mac Editor 中无法使用。
  • WG-25733：无法使用 Windows 编辑器构建 iOS/Mac 游戏，反之亦然。
