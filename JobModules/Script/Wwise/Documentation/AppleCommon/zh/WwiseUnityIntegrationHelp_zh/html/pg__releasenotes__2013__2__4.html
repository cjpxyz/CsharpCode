版本说明 - Wwise Unity Integration 2013.2.4

此次发布为 Wwise 2013.2.4 提供了 Wwise SDK 更新，修复了错误，做了一些改进。

兼容性：

  • Wwise SDK: 2013.2, 2013.2.x
  • Unity: 4.x

支持的平台：

  • Windows 8（Metro）
    • Desktop
    • 在 Intel 设备上的 Windows Store App，和 ARM 平板电脑
  • Windows
    • 32 位
    • 64 位
  • Mac OS X（10.6 和更高版本）
  • iOS
  • Android
  • Xbox360
  • PS3

新功能：

  • 新菜单项目 Wwise > Help，用于在默认的网页浏览器中打开 Integration 文档。
  • 添加 Version.txt 元数据文件给所有发布，以指明基本的 Wwse SDK信息。

已解决的问题：

  • WG-24080：在 Windows 和 Mac API 切换到 Android 之后出现的编译器错误。

迁移：

  • 移除 Android armeabi 插件。
