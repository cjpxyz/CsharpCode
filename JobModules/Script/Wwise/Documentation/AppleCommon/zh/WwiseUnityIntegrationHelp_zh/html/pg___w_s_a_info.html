关于 Windows Store Apps 的特殊信息

  • Universal Windows Platform 使用普通的 Windows SoundBank。The SoundBank platform folder is the same for both of these platforms.
  • It is not possible for Wwise to communicate with a Windows Store App that is running on the same machine.
  • To enable communication between Wwise and an App, the "PrivateNetworkClientServer" capability needs to be set. This can be done by selecting the following menu item: Assets > Wwise > WSA > Enable PrivateNetworkClientServer Capability
