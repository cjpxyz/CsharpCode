详细描述

Use this component to define a reverb zone. This needs to be added to a collider object to work properly. 如何通过 inspector来使用 AkEvironment 和 AkEvironmentPortal（混响区域）.

This component can be attached to any collider. You can specify a roll-off to fade-in/out of the reverb. The reverb parameters will be defined in the Wwise project, by the sound designer. All AkGameObj that are "environment"-aware will receive a send value when entering the attached collider.
