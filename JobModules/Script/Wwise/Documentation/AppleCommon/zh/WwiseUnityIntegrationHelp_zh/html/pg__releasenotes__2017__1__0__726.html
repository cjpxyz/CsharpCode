版本说明 - Wwise Unity Integration 2017.1.0.6302.726

本次发布为 Wwise 2017.1.0 提供了 Wwise SDK 更新，并与 Unity 2017.1 兼容。

  • 新功能 2017.1
  • 重要迁移说明 2017.1

兼容性：

  • Wwise SDK: 2017.1.0
  • Unity: Unity 2017.1（Personal 或 Pro）。
    		备注： 重要提示：该集成还没有在 Unity 4 上测试过。
    平台： Wwise Unity Integration 在以下平台测试过：
    Mac
    Windows

    Android
    iOS

    PS4
    PS Vita
    tvOS
    UWP (Universal Windows Platform)
    Xbox One     		Unity 5.6.1p1
    Switch Unity for Switch 1.0.1 (Unity 5.5.0p1 和 Nintendo SDK 1.4.0)

新功能：

  • 添加了一些新功能，详情请参见 新功能 2017.1 页面。

重要迁移说明：

  • 不再支持 Wii U 平台。
  • 包含自动注册 GameObject 的功能已经被转移到 Unity Integration Extensions。关于此更改更详细的信息，请参见 Wwise Installation and Migration Guide。
  • 针对 AkCallbackManager 有许多重大更改，详情请见 重要迁移说明 2017.1 页面。

已解决的问题：

  • WG-27479：可以通过 AkInitializer 配置回调缓冲区大小。
  • WG-30791：添加了 WwiseTypes。详情参见 Wwise Types。
  • WG-31155：修复了 AkMemSettings 和 AkChannelConfig，因此它们的方法不检查 SoundEngine 初始化。
  • WG-31735：添加了钩子，使用户可以方便地扩展或删除 GameObjects 的自动注册。详情参见 Unity Integration Extensions（集成扩展）。
  • WG-32348：修复了 SoundBanksInfo 文件的 XML 解析。
  • WG-32657：在使用回调类时减少内存分配，并删除 foreach 语句。
  • WG-33303：为 AK_MusicPlaylistSelect 添加支持。
  • WG-34003：安装过程不再会禁用工程中的其它原生插件。
