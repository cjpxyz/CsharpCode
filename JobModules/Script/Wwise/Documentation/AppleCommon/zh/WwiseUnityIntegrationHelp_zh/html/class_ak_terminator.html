AkTerminator类参考

详细描述

This script deals with termination of the Wwise audio engine. It must be present on one Game Object that gets destroyed last in the game. It must be executed AFTER any other monoBehaviors that use AkSoundEngine.

参见:
  • Terminate the Different Modules of the Sound Engine (Note: This is described in the Wwise SDK documentation.)
  • AK::SoundEngine::Term() (Note: This is described in the Wwise SDK documentation.)
