版本说明 - Wwise Unity Integration 2017.2.8.6698.1171

此次发布为 Wwise 2017.2.8 提供了 Wwise SDK 更新。

兼容性：

  • Wwise SDK: 2017.2.8
  • Unity: Unity 5.6, 2017.1, 2017.2, 2017.3, 2017.4, 2018.1, 和 2018.2 (Personal 或 Pro)。

    Android

已解决的问题：

  • WG-38413 Fixed: Pausing the editor then entering play mode causes Unity to freeze.
  • WG-39195 Fixed: Spamming of the editor listener position and orientation when values are constant.
  • WG-39201 Removed error message when adding an AkBank component from the Add Component button.
  • WG-39235 Added log message for when the sound engine is already initialized.
  • WG-39307 Made attenuation sphere visualization 90% transparent.
  • WG-39388 Fixed: Ak Initializer settings are not persisted on project reload
