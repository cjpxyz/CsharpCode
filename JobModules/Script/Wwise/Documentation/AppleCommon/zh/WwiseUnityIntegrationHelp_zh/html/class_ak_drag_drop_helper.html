AkDragDropHelper类参考

This class is used to perform DragAndDrop operations from the AkWwisePicker to any GameObject. We found out that DragAndDrop operations in Unity do not transfer components, but only scripts. This prevented us to set the name and ID of our components before performing the drag and drop. To fix this, the DragAndDrop operation always transfers a AkDragDropHelper component that gets instantiated on the target GameObject. On its Awake() call, it will parse the DragAndDrop structure, which contains all necessary information to instantiate the correct component, with the correct information. 更多...

This class is used to perform DragAndDrop operations from the AkWwisePicker to any GameObject. We found out that DragAndDrop operations in Unity do not transfer components, but only scripts. This prevented us to set the name and ID of our components before performing the drag and drop. To fix this, the DragAndDrop operation always transfers a AkDragDropHelper component that gets instantiated on the target GameObject. On its Awake() call, it will parse the DragAndDrop structure, which contains all necessary information to instantiate the correct component, with the correct information.
