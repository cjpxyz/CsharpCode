版本说明 - Wwise Unity Integration 2015.1.3

本次发布为 Wwise 2015.1.3 和 Universal Windows Platform 提供了 Wwise SDK 更新。

兼容性：

  • Wwise SDK: 2015.1.3
  • Unity：Unity 4.6 Pro 或 Unity 5.2 (Personal 或 Pro)
    平台： Wwise Unity Integration 在以下平台测试过：
    Android
    iOS
    Linux
    Mac
    PS3
    PS4
    PS Vita
    Windows
    Windows Store Apps
    Xbox One     		Unity 5.2.2p1
    Wii U Unity 4.3.7f1 和 Wii U add-on 2.2.5
    Xbox 360 Unity 5.0.1f1

新功能：

  • 在 Windows Store Apps 包下添加了对 Universal Windows Platform 的支持。
  • 添加了对 Windows Store Apps SDK 8.1 的支持。
  • 取消了对 Windows Phone 8.0 的支持。
  • 取消了对 Windows Store Apps SDK 8.0 的支持。

已解决的问题：

  • WG-25945 修复了：Wwise 错误总是被重定向到 Unity 控制台。在 WwiseGlobal 设置中添加一个菜单项，以便在 Unity 控制台中启用/禁用 Wwise 引擎输出。
  • WG-26011 修复了：在创建一个新场景后，启用回 Ak Audio Listener 不能正确地破坏 Default Unity Audio Listener。
  • WG-28108 修复了：当暂停编辑器时，声音仍然一直在播放。现在在编辑器中暂停后，声音会停止；取消暂停后，声音会重新播放。声音在切换窗口时（和以前一样）一直在播放。
  • WG-28175 修复了：在创建新场景时，WwiseGlobal 并不是每次都自动创建。
  • WG-28479 修复了：安装时尝试添加未勾选的选项。 现在在没有 Main Camera 的工程中安装集成时，安装会成功。
  • WG-28526 修复了：潜在的 Unity Editor 崩溃。确保了在注册和取消注册 GameObjects 之前，先对 SoundEngine 进行了初始化。
