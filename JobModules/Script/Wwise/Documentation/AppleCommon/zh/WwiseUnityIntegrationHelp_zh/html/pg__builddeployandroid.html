为 Android 平台构建应用程序

APK 中包含的 Android 的 SoundBank 可以直接通过 LoadBank() API 访问。当构建 Unity 工程时，放置在 StreamingAssets 文件夹中的 SoundBank 会包含在该 APK 的 Assets 文件夹中。Low-Level IO 可以透明地从 APK 中访问 SoundBank。

Low-Level IO 也可以访问 Android 文件系统中的 SoundBank 文件，例如访问 SD 卡中的文件。这不是默认行为。如果您想要访问原生文件系统，则需要调用 AkSoundEngine.AddBasePath(路径名)。这条新路径将比默认的 APK 中的文件有更高优先级。如果一个文件在外部路径和 APK 中具有相同的名称，那么将使用外部路径。这对于在构建 APK 之后，实现 DLC 或者实现对音频内容的补丁是很有用的。
