第一次安装 Wwise 集成包

集成过程由 Wwise Launcher 负责。请使用 Wwise Launcher 来将 Wwise 集成到 Unity 工程中。

警告： 在开始之前：
  • 确保您已经备份了 Unity 工程，要么通过版本控制，要么通过 Launcher 中的备份功能。
  • 在安装 Wwise 集成之前，需要确保您的 Unity 工程可以成功编译。（出现警告是正常的。）
  • 如果 Unity 工程有版本控制，那么在开始集成之前一定要 check out 整个 Unity 工程文件夹。否则，将无法集成，因为无法修改文件。

Unity Integration Extensions（集成扩展）

以 partial 方法的形式添加了用户钩子，这样用户可以扩展游戏对象的自动注册功能或将此功能全部删除。尽管我们推荐使用默认的扩展，但该扩展可以通过 Launcher 来选择省略掉。
