版本说明 - Wwise Unity Integration 2013.2.5

此次发布为 Wwise 2013.2.5 提供了 Wwise SDK 更新，对若干新平台的支持，和对错误的修复。

兼容性：

  • Wwise SDK: 2013.2, 2013.2.x
  • Unity: 4.x

支持的平台：

  • Android
  • iOS
  • Mac OS X（10.6 和更高版本）
  • PS3
  • PS4
  • PS Vita
  • Windows
    • 32 位
    • 64 位
  • Windows 8（Metro）
    • Desktop
    • 在 Intel 设备上的 Windows Store App，和 ARM 平板电脑
  • Xbox 360
  • Xbox One

新功能：

  • 添加了对新平台的支持：PS4。
  • 添加了对新平台的支持：PS Vita。
  • 添加了对新平台的支持：Xbox One。
  • Android：添加了一个新 API AddBasePath()，用于为 I/O 指定一个 POSIX 文件系统路径。

已解决的问题：

  • WG-24351：iOS：没有处理远程控制中断。
