版本说明 - Wwise Unity Integration 2014.1.5

此次发布为 Wwise 2014.1.5 提供了 Wwise SDK 更新。 兼容性：

  • Wwise SDK: 2014.1.5
  • Unity：Unity 4.6.5 Pro, 或 Unity 5.0.2 (Personal 或 Pro)
    平台： Wwise Unity Integration 在以下平台测试过：
    Android
    iOS
    Linux
    Mac
    Metro
    PS3
    PS4
    PSVita
    Windows
    Windows Phone     		Unity 5.0.2p3
    Xbox 360 Unity 5.0.1f1
    Xbox One Unity 5.0.2p4

已知问题：

  • WG-25669 Fixed: Auto Populate of the Wwise Picker now works in the Mac Editor.

已知的 Unity 问题：

  • Xbox One：SoundEngine 初始化失败，即使 Unity 音频被禁用。请参见 故障排除。
  • Windows Store Apps in Unity 4：构建的游戏会抛出一个 DllNotFoundException。请参见 故障排除。

新功能：

  • 添加了对 Android x86 的支持。
  • 对 Windows Store Apps 平台，消除了对 Scripting Define Symbols 的需求。

已解决的问题：

  • WG-25733 修复了：现在可以使用 Windows 编辑器来构建 iOS/Mac 游戏，反之亦然。
  • WG-26655 iOS: Fixed: Audio now restores after phone Interruption on iOS 8 under AmbientSound category.
  • WG-26875 修复了：现在 AkMemBankLoader 组件分配了对齐的内存，防止在某些平台上出现故障。
  • WG-27108 Fixed: Now there are no Unity error messages when posting events during the Destroy phase.
  • WG-27461 Fixed: AkGameObj set on the same object as AkEnvironment now gets the environment.
