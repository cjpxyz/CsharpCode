Wwise Unity 集成

Wwise Unity 集成 是一个 Unity 插件，它能让我们在 Unity 游戏中使用 Wwise Sound Engine。在文档后面的部分中，我们将称它为 "集成"（Integration）。集成包含一个 Unity 插件库、用 C# 写的 Wwise SDK API 绑定、一些有用的组件和 Editor 窗口。如果要了解 Wwise SDK 的基本功能，请参阅其文档，文档可以 在线阅读，也可以在您的 Wwise 安装找到它的 CHM。

这个集成文档提供了关于 Unity-Wwise 集成和 Wwise API 的信息。如果您需要关于 Wwise 设计工具的信息（对于声音设计师），请参考正确的帮助文件，可以在 在线文档 中找到，也可以在您的本地安装中找到（一般在 C:\Program Files(x86)\Audiokinetic\Wwise [version] \Authoring\Help 中）。
